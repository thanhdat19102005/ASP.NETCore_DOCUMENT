5	Quần	Thời trang	4000

7	Giày	Thời trang	2500
📌 Yêu cầu:

Lấy các nhóm Category có giá > 1000 (lọc bản ghi trước).

Gom nhóm theo Category.

Chỉ lấy nhóm có từ 2 sản phẩm trở lên (lọc nhóm sau).

Xuất Category + số lượng sản phẩm.

🔹 SQL
SELECT Category, COUNT(*) AS SoLuong
FROM Items
WHERE Price > 1000              -- Lọc bản ghi (WHERE)
GROUP BY Category                -- Gom nhóm
HAVING COUNT(*) >= 2             -- Lọc nhóm (HAVING)
ORDER BY SoLuong DESC;           -- Sắp xếp

🔹 LINQ Method Syntax
var query = Items
    .Where(item => item.Price > 1000)              // WHERE
    .GroupBy(item => item.Category)                // GROUP BY
    .Where(g => g.Count() >= 2)                    // HAVING
    .OrderByDescending(g => g.Count())             // ORDER BY
    .Select(g => new { Category = g.Key, Count = g.Count() }); // SELECT lấy  gia  những cột mình  muốn
