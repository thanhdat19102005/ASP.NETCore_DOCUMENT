   gameObject.TryGetComponent<T>(out T component )    :  dùng để kiểm tra xem GameObject đó có component kiểu T hay không và  giá trị 
trả về  là  true or false   nếu false T component  = null  

    giải thích    T  : tên  Component  
                 T component :   Đây là tên biến bạn đặt để lấy   component  của gameObject trả về true  

  


    Ví dụ  : gameObject.TryGetComponent <PolygonCollider2D> (out PolygonCollider2D lp) 
             lp  .istrigger   = true  ;
