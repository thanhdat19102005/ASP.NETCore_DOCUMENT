namespace  là  gì  ?  
namespace = cái hộp / thư mục logic trong code, giúp quản lý, phân loại, và tránh trùng tên class.



Một namespace có thể nằm ở nhiều file khác nhau.

📌 Ví dụ:

📂 Project của bạn có 2 file:

📄 PlayerMovement.cs

namespace MyGame.Player
{
    public class PlayerMovement
    {
        public void Move()
        {
            Console.WriteLine("Player is moving!");
        }
    }
}


📄 PlayerAttack.cs

namespace MyGame.Player
{
    public class PlayerAttack
    {
        public void Attack()
        {
            Console.WriteLine("Player is attacking!");
        }
    }
}


👉 Cả 2 class PlayerMovement và PlayerAttack đều thuộc namespace MyGame.Player, dù chúng nằm ở 2 file khác nhau.
