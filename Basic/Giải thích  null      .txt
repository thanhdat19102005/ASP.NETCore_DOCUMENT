Giải thích  null    :     null = không tồn tại giá trị, không tham chiếu đến đối tượng nào.


I >  kiểu dữ liệu là  thuộc tính [fiel / property]  : thì IDE sẽ tự động gán giá trị mặc định cho thuộc tính [fiel /  property] và Grogrammer ko cần làm việc đo  

 *   thuộc tính   [fiel /  property]    nếu   khai báo  không  gán  giá trị  thì   IDE  sẽ tự động gán NULL cho thuộc tính   [fiel /  property]
* thực chất  kiểu dữ liệu tham  chiếu  IDE tự động  gán  Null   . còn nguyên thủy (long long ,  double  , float ,  int )  IDE tự động  gán  0 ,boolean IDE tự đông gán false  nên vậy muốn   
   

 public class SinhVien {
    public  int a { get; set; }   //   a  =  0    ||      public   int ?   a   {get;set;}   a = null 
}

thì sử dụng cấu trúc
